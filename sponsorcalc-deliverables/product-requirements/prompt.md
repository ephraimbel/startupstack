# SponsorCalc - Product Requirements Document

## Core Features

### Social Account Connection
**Priority**: P0 (Critical)
**Description**: OAuth integration for Instagram, YouTube, TikTok to pull real engagement data
**User Story**: As a creator, I want to connect my social accounts so the platform can analyze my real performance data
**Requirements**: Secure OAuth, data privacy compliance, error handling for API changes

### Rate Calculation Engine  
**Priority**: P0 (Critical)
**Description**: AI-powered algorithm analyzing engagement, demographics, and niche CPMs
**User Story**: As a creator, I want accurate rate calculations based on my actual performance, not just follower count
**Requirements**: Multi-factor algorithm, real-time data processing, transparent methodology

### Professional Rate Card Export
**Priority**: P0 (Critical)  
**Description**: Generate branded PDF rate cards suitable for brand outreach
**User Story**: As a creator, I want professional rate cards I can confidently send to any brand
**Requirements**: Multiple templates, mobile-generated PDFs, brand customization options

### Pricing Tier Management
**Priority**: P0 (Critical)
**Description**: Freemium model with clear feature gates and upgrade paths
**User Story**: As a user, I want to try the platform for free and upgrade when I see value
**Requirements**: Stripe integration, usage tracking, clear upgrade prompts

## Feature Specifications

### MVP Features
1. **Social Platform Connection**: Instagram, YouTube, TikTok APIs
2. **Manual Platform Entry**: Podcasts, newsletters, other platforms
3. **Rate Calculation**: Multi-factor algorithm with niche CPM data
4. **Tiered Rate Cards**: Low/medium/high pricing for different deliverables
5. **PDF Export**: Professional branded templates
6. **Industry Benchmarks**: Anonymous comparison data
7. **Negotiation Scripts**: Pre-written responses to common objections
8. **Deal Tracker**: Log past deals and outcomes

### Post-MVP Features
- Media kit builder beyond just rates
- Brand database with typical budgets  
- Rate trend tracking over time
- AI pitch writer for brand outreach
- Contract templates and legal review
- Invoice generation and payment tracking
- Team/agency dashboard for multiple creators
- API access for third-party integrations
