# RepurposeAI Brand Package - AI Generation Prompt

## Overview
Create a comprehensive brand identity system for RepurposeAI, an AI-powered content repurposing platform that helps creators turn one video into multiple platform-specific content pieces.

## Brand Positioning
- **Product**: AI tool that transforms YouTube videos/podcasts into Twitter threads, LinkedIn posts, TikTok clips, newsletters, etc.
- **Target Audience**: Content creators (YouTubers, podcasters, course creators) with 10K+ following who publish weekly
- **Core Value Prop**: "One video. Endless content." - Turn 4+ hours of repurposing work into 10 minutes
- **Personality**: Energetic ally for creators, like a successful creator friend sharing a growth hack

## Generate the Following Brand Elements:

### 1. Brand Name Variations
- Primary: RepurposeAI
- Alternative taglines to test:
  - "One video. Endless content."
  - "Turn content into content machines."
  - "Your AI content multiplier."

### 2. Logo Concepts (Describe for Designer)
**Primary Concept**: Abstract 'play' button that fragments into multiple smaller shapes, representing one piece of content becoming many
- Style: Modern, clean, tech-forward
- Colors: Gradient from violet (#7C3AED) to coral (#F97316)
- Format: Horizontal wordmark + icon, stacked version, icon-only

**Alternative Concepts**:
- Content multiplication symbol (1→∞)
- Geometric content flow diagram
- Video frame transforming into social media icons

### 3. Color Palette
```
Primary: #7C3AED (Creator Violet) - Buttons, key UI, logo
Secondary: #1E1B4B (Midnight Ink) - Headlines, text
Accent: #F97316 (Engagement Orange) - CTAs, highlights, notifications
Neutral Light: #F8FAFC (Canvas White) - Backgrounds
Neutral Dark: #334155 (Slate) - Secondary text
```

### 4. Typography System
- **Headlines**: Cal Sans / Space Grotesk (modern, friendly)
- **Body Text**: Inter (clean, readable)
- **UI Elements**: Inter (consistency)

### 5. Brand Voice & Tone
- **Personality**: Energetic but not manic, knowledgeable creator ally
- **Tone**: Like a successful creator friend excited to share a growth hack
- **Language**: Creator-native (algorithm, engagement, hooks, viral)
- **Avoid**: Corporate jargon, overpromising viral results

### 6. Brand Applications
Generate copy for:
- Email signature
- Social media bios (Twitter, LinkedIn, YouTube)
- One-line boilerplate for press
- Tagline variations for different contexts

### 7. Visual Style Guidelines
- **Photography**: Real creators in natural environments, diverse representation
- **Illustrations**: Simple, geometric, showing transformation/multiplication
- **UI Elements**: Rounded corners, subtle shadows, gradient accents
- **Icons**: Outline style, consistent weight

### 8. Brand Story (2-3 paragraphs)
Tell the founding story focusing on:
- Creator frustration with manual repurposing
- Discovery that AI could solve this
- Vision of empowering solo creators to compete with teams

## Deliverable Format
Provide as a comprehensive brand guideline document including:
1. Logo files (SVG, PNG at various sizes)
2. Color codes (HEX, RGB, CMYK)
3. Font specifications with download links
4. Brand voice examples (do's and don'ts)
5. Application examples across different mediums
6. Brand story narrative

## Technical Notes for Implementation
- Ensure logos work at favicon size (16x16px)
- Test color contrast for accessibility (WCAG AA)
- Provide both light and dark mode versions
- Include responsive logo variants

This brand should feel premium but approachable, tech-savvy but human, energetic but trustworthy - perfectly positioned for the creator economy market.
